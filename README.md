# Mood Tracker Application

A full-stack web application which tracks the user's daily moods.

## Tech stack
* Frontend: React, Tailwind CSS
* Backend: Node.js, Express
* Storage: JSON files

## Features (Planned)
* Daily mood logging with intensity scale
* Daily journal entry paired with mood log
* Data persistence

## Setup Instructions

**Install dependencies:**
```bash

# Client
cd client
npm install

# Server
cd ../server
npm install
```

## Running the Application
**Terminal 1 - Backend:**
```bash
cd server
npm run dev
```

**Terminal 2 - Frontend:**
```bash
cd client
npm run dev
```
